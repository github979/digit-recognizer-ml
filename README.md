# 🧠 Digit Recognizer (ML → DL Evolution)

## 🚀 Overview
This project predicts handwritten digits (0–9) using the MNIST dataset from Kaggle.

It showcases the progression from a basic Machine Learning model to a Deep Learning model.

---

## 📊 Dataset
- Source: Kaggle Digit Recognizer
- Images: 28x28 grayscale (784 pixels)
- Labels: Digits from 0 to 9

---

## ⚙️ Models Used

### 🔹 Logistic Regression (Baseline)
- Simple ML model
- Accuracy: **0.91**

📁 File: `digit-recognizer-ml.ipynb`

---

### 🔥 Convolutional Neural Network (CNN)
- Deep Learning model for image recognition
- Accuracy: **0.98**

📁 File: `digit-recognizer-cnn.ipynb`

---

## 📈 Results

| Model                | Accuracy |
|---------------------|---------|
| Logistic Regression | 0.91    |
| CNN                 | 0.98 🚀 |
